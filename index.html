Smart Brain

Smart Brain is a full-stack web application designed to perform facial recognition using the Clarifai API, enabling users to upload images and detect faces with ease. Built with JavaScript, React, Node.js, and PostgreSQL, the application offers a responsive, intuitive interface on the front end while maintaining robust server functionality and secure data handling on the back end.

AI Fitness Chatbot

I developed a fitness chatbot that acts as a personal fitness assistant using Nuxt.js, and provides users with personalized workout, nutrition or any fitness related answers or advice that the user is looking for to help them reach their goals. Using the power of the OpenAI API, the chatbot interacts with users to deliver tailored responses based on their unique fitness goals, current activity levels, and personal preferences..

Pathfinding Algorithm Visualizer

This Pathfinding Algorithms Visualizer is a React.js application that helps demonstrate how two pathfinding algorithms work in real time.In the maze we can explore and compare the efficiency of algorithms like breadth-first search (BFS) and depth-first search (DFS) through randomized mazes. The visualizer offers a clear, dynamic display of optimal paths and unique maze-solving strategies, with smooth animations and responsive rendering.
